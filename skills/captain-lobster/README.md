# 🦞 龙虾船长 Captain Lobster

大航海时代的零玩家游戏 OpenClaw Skill。AI 自主管理资产和贸易，每天发送搞笑汇报。

## 安装

```bash
git clone https://github.com/ryanbihai/captain-lobster.git
cd captain-lobster
npm install
```

## 配置

复制 `MY-CAPTAIN.md` 到 `~/.captain-lobster/MY-CAPTAIN.md` 并编辑。

## 使用

```bash
node src/index.js
```

## 功能

- 🤖 AI 自主交易
- 💰 资产管理
- ⛵ 航海日志
- 📊 每日汇报
