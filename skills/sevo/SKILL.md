# SEVO Pipeline

Agent 自动研发流水线，覆盖 Specify → Plan → Implement → Review → Publish 全生命周期。

## 安装

```bash
npm install sevo-pipeline
```

## 核心能力

- 8 阶段研发流水线（需求定义 → 架构设计 → 编码 → 审计 → 发布）
- OKR 目标声明 → SMART 任务拆解 → PDCA 差距分析
- 终局交付自动化（Endgame Delivery Stage）
- 阶段门禁与质量审计
- 可编程宿主 API（PluginAdapter / NotificationAdapter）
