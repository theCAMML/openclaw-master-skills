# 股市晨報 Skill（v1.3.9）

## I 報告格式

```
🔴股市晨報(x.xx) 🔵 | 標題1；標題2；標題3

「x內部文件 不得外傳」

📍一、[昨日/週X]股市回顧
[市場概況 + 強弱勢股]

📍二、南下資金凈買入XX億（港元）
📈 港股通(滬): 凈買入XX億
📉 港股通(深): 凈賣出XX億
其中：凈買入[股票]XX億；凈賣出[股票]XX億

📍三、熱點資訊
▶️ 標題
摘要（空一行）
▶️ 標題
摘要（空一行）
▶️ 標題
摘要（空一行）

📍四、熱門港股
▶️ [股票名稱+代碼]
摘要

資料來源: xxx
「只供內部參考，不可外傳」
```

## II 格式規則

2.1 大標題 = 月份.日期 + ▶️分號連接

2.2 📍後無空格，直接接字

2.3 （港元）：小標題行有；📈📉和個股明細無

2.4 ▶️ 標題↔摘要：空一行（`\n\n`，不是`&nbsp;`）

2.5 港股通(滬)↔港股通(深)：無空行，緊密排列

2.6 第四部分只寫1檔

2.7 搜不到數字 → 擴大搜索範圍、更換關鍵詞、換數據源，**不存在「待更新」這個退出選項**

## III 執行流程

3.1 確認今日日期與上一交易日，查gov.hk確認是否交易日

3.2 讀 `references/stock_report_format.md` 取格式

3.3 抓恒指/科指（騰訊財經API）

3.4 搜市場回顧（gelonghui.com）

3.5 搜南下資金（stcn.com全市場口徑），同時搜個股明細

3.6 搜第三部分（今日日期的新聞，不是舊聞）

3.7 搜第四部分（「熱門股票」主題，選1檔）

3.8 按模板組裝，大標題=▶️分號連接

3.9 核查9項（見下方）後發送

3.10 飛書只發卡片

**⚠️ 每日必生成原則（2026-05-03 強化）：**
- **所有日子都生成報告，包括週末、假期、補假**
- 交易日：生成標準五部分格式晨報
- 非交易日（週末/假期/補假）：生成「宏觀特報」，格式見下方第八節
- **禁止跳過任何一天**，即使港股休市也必須生成並發送

## IV 發送前核查（9項）

4.1 [ ] 大標題 = ▶️分號分隔

4.2 [ ] 📈📉方向正確（📈=凈買入）

4.3 [ ] （港元）只在正確位置

4.4 [ ] ▶️↔摘要有空行

4.5 [ ] 📍後無空格

4.6 [ ] 第四部分1檔

4.7 [ ] 數字有靠譜來源

4.8 [ ] 📈📉數字：必須找到真實來源，**不得以任何理由寫待更新**

4.9 [ ] 個股明細已完整填充

## V 禁區

5.1 第一部分不準猜測假日 → 先查gov.hk

5.2 早上不引用當天收盤數 → 只用上一交易日

5.3 數字不確定 → 繼續搜，**寫待更新等於承認失敗**

5.4 修復報告時重新生成全報告 → 最小修改原則

5.5 個股明細留空 → 繼續搜完，**不得以待更新敷衍**

5.6 **對話 session 轉發原則**：收到「生成晨報」請求時，先檢查 `stock_morning_report.md` 是否存在且當日生成。若存在，直接讀取轉發，不重新搜索。若不存在，才走完整流程生成。
